• Mouse Events
    1. click: Triggered when a user clicks an element.
    2. dblclick: Triggered when a user double-clicks an element.
    3. mousedown: Triggered when a mouse button is pressed down over a element.
    4. mouseup: Triggered when a mouse button is released over an element.
    5. mousemove: Triggered when the mouse cursor moves over an element.
    6. mouseover: Triggered when the mouse pointer enters an element.
    7. mouseout: Triggered when the mouse pointer leaves an element.
  • Keyboard Events
    1. keydown: Triggered when a key is pressed down.
    2. keypress: Triggered when a key that produces a character is pressed down.
    3. keyup: Triggered when a key is released.
  • Form Events
    1. focus: Triggered when an element gains focus (e.g., input field selected).
    2. blur: Triggered when an element loses focus.
    3. change: Triggered when the value of an element changes (e.g., after selecting a dropdown option or typing in an input field and moving focus away).
    4. input: Triggered when the value of an input element changes.
    5. submit: Triggered when a form is submitted.
    6. reset: Triggered when a form is reset.
  • Window Events
    1. load: Triggered when a page or an element (like an image) has finished loading.
    2. scroll: Triggered when the user scrolls the page or an element.
    3. resize: Triggered when the browser window is resized.
    4. beforeunload: Triggered before the window is unloaded (e.g., when closing the tab or navigating away).
  • Document Events
    1. DOMContentLoaded: Triggered when the initial HTML document has been completely loaded and parsed, without waiting for stylesheets, images, and subframes to finish loading.
  • Touch Events (for mobile devices)
    1. touchstart: Triggered when a touch point is placed on the touch surface.
    2. touchmove: Triggered when a touch point is moved along the touch surface.
    3. touchend: Triggered when a touch point is removed from the touch surface.
    4. touchcancel: Triggered when a touch interaction is interrupted (e.g., by a system event).
